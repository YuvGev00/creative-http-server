loom/http

A tiny http/1.1 framework woven from nothing but node's net, fs, path, and one hash from crypto. Express-shaped api, typed routes with auto-validation, live docs that can't drift, a flight recorder, hand-rolled websockets, and a multiplayer draw-and-guess game — all on the same raw sockets.
